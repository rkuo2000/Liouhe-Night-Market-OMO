# 六合夜市 線上點餐系統

## 系統功能
1. 手機掃描QR code進入網頁 (無須安裝App)
2. 提供各攤位專用線上點餐之頁面
3. 支援各國語音及文字的選項
4. 點選美食照片，提供各國語音及文字做餐點說明
5. 點選數量
6. 計算總金額及產生餐點訂單
7. 支援Line傳送
